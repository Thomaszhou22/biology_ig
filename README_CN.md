## 🌍 项目使命

**让全球 CIE 学生都能免费复习生物。** Biology 是一个互动式复习工具，让 IGCSE Biology (0610) 的学习变得有趣且易于获取——无需注册、无需付费、无需安装，打开即玩。

由学生开发，为学生服务。

## ✨ 功能

### 📚 完整题库（777 题）

涵盖 IGCSE Biology 全部 21 个章节，共 777 道图片选择题。

### 🎮 游戏模式
- **章节练习** — 掷骰子、答题、竞速通关
- **单人 / 双人** 模式
- **章节选择** — 自由组合章节练习
- **图片题目** — 每道题都有清晰配图

### 📖 学习工具
- **错题本** — 自动收录错题，方便复习
- **即时反馈** — 答题后立即显示正确答案
- **进度保存** — localStorage 自动保存，随时继续
