# 参与 Biology — IGCSE Edition 贡献 🐸

首先，感谢你考虑为本项目做贡献！本项目的目标是帮助全球 CIE IGCSE Biology 学生免费复习。

## 如何参与

### 📝 添加或修正题目
- 题目以图片形式存储在 `questions/chXX_topicname/` 目录下（如 `questions/ch18_variation/q01.jpg`）
- 每个章节文件夹包含编号的题目图片
- 添加题目：在对应章节文件夹中添加新的编号图片
- 修正题目：替换错误的图片文件

### 🌍 翻译界面
- 目前游戏界面为英文
- 欢迎翻译成其他语言（中文、西班牙语、阿拉伯语等）
- 开始翻译前请先开 Issue 讨论

### 🐛 报告问题
- 发现答案有误？图片加载失败？界面出错？
- 提交 [GitHub Issue](../../issues)，包含：
  - 章节和题号
  - 问题描述
  - 截图（如有）

### 💡 功能建议
- 使用 `enhancement` 标签提交 Issue
- 描述功能及其对学生学习的作用

## 贡献原则

1. **保持免费** — 这是一个教育公益项目
2. **准确性优先** — 提交前请仔细核对生物学内容
3. **友善交流** — 这是一个学生项目，请提供建设性反馈

## 提交更改

1. Fork 本仓库
2. 创建功能分支（`git checkout -b feature/your-feature`）
3. 提交更改
4. 发起 Pull Request

感谢你帮助学生更好地学习！🎓
