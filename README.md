# 🧬 Biology IG — IGCSE 生物选择题练习

Cambridge **IGCSE Biology (0610)** 选择题练习工具。涵盖全部 21 个章节，共 **744 道**题，支持章节筛选、错题本、即时反馈。

🌐 **在线练习**: [thomaszhou22.github.io/biology_ig](https://thomaszhou22.github.io/biology_ig/)

### 🎯 章节选择
- 自由组合任意章节进行练习
- 支持全选 / 取消全选
- 可设置每章抽取的题目数量

### ✅ 答题功能
- **即时判分** — 选择后立即显示对错，高亮正确答案
- **答题计时** — 每道题自动计时
- **进度追踪** — 实时显示正确 / 错误 / 总题数和进度条
- **移动提示** — 可选显示每回合允许移动的格数

### 📖 错题本
- 自动收录答错的题目
- 支持查看和重做错题，针对性复习

### 🔒 管理员模式
- 隐藏管理面板，通过特殊触发方式进入
- 支持答题测试模式切换

- **无需安装**: 浏览器打开即用
