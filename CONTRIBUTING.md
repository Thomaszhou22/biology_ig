# Contributing to Biology — IGCSE Edition 🐸

First off, thank you for considering contributing! This project exists to help CIE IGCSE Biology students worldwide study for free.

## How You Can Help

### 📝 Add or Fix Questions
- Questions are stored as images in `questions/chXX_topicname/` (e.g., `questions/ch18_variation/q01.jpg`)
- Each chapter folder contains numbered question images
- To add questions: add new numbered images to the appropriate chapter folder
- To fix a question: replace the incorrect image file

### 🌍 Translate the Interface
- The game interface is currently in English
- We'd love translations into other languages (Chinese, Spanish, Arabic, etc.)
- Open an issue to discuss before starting a translation

### 🐛 Report Issues
- Found a wrong answer? Image not loading? UI broken?
- Open a [GitHub Issue](../../issues) with:
  - Chapter and question number
  - Description of the problem
  - Screenshot if applicable

### 💡 Suggest Features
- Open an issue with the `enhancement` label
- Describe the feature and why it would help students

## Guidelines

1. **Keep it free** — this is an educational公益 project
2. **Accuracy first** — double-check biology content before submitting
3. **Be respectful** — this is a student project, constructive feedback only

## Submitting Changes

1. Fork this repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes
4. Open a Pull Request

Thank you for helping students learn! 🎓
